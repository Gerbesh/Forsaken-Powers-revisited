# Changelog

## 1.0.0

- Initial public release
- Strict boss-based Forsaken Power unlock validation
- Fixed availability of **Fader** power in worlds where the boss was not defeated
- Server-authoritative activation logic
- Alt-key activation of all unlocked powers (server-side only)
- Configurable cooldowns, durations and messages
- Passive mode support
- No external sync libraries (ServerSync / ConfigSync removed)
- Improved safety for public servers
